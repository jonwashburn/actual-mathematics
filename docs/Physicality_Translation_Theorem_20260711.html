Frozen contract, 2026-07-11

Physicality Translation Theorem

The theorem closes one precise capacity: a δ-kernel theorem whose checker spent nothing can be read as true in every admitted Peano realization, with the original checker equation still attached. Calling that target truth physical requires a separate empirical receipt supplied from outside the proof kernel.

Accepted scope. The formal result is a fixed-signature semantic transport theorem for DFormula. It does not yet translate arbitrary mathematical signatures, and it does not prove that nature supplies a realization.

First-principles brief

Why

The δ bootstrap now has two pieces that should meet: proof-relevant construction history and choice-free transport into every Peano model. Without a frozen contract, “math becomes physical truth” admits fake proofs based on a target semantics that declares everything true, an empty model family, or a receipt written by the translator itself.

Who

The immediate audience is a mathematician or proof tool that needs to know exactly which theorem, construction, semantic model, and empirical premise are being transported. The larger audience is the Physicality Grammar, where each deposited edge must eventually carry the same information.

Goal

Define the smallest institution-style kernel that preserves and reflects satisfaction, package computed checker receipts and construction receipts, require a nonempty externally attested target family, and prove the concrete δ transport theorem.

Success

Lean builds the new modules. Forced addition commutativity reaches both the natural and list carriers. Seven hostile formulations fail for named reasons. Axiom audits show that the forced transport core adds no Classical.choice.

Constraints

Existing Lean statements are authoritative. The target must prove the Peano laws. The empirical bridge is a premise. The original checker equation is retained verbatim. No theorem may inhabit the empirical existence target.

Premises challenged before implementation

  1. What is the strongest reason the framing is wrong? The current semantic kernel speaks one arithmetic language, DFormula. It cannot prove a theorem about all mathematical languages. This implementation closes the arithmetic instance and exposes the interface that later signatures must inhabit.
  2. What is the smallest artifact that creates the capacity? One satisfaction-equivalence translation, one model-expansion theorem, one externally attested family type, and one δ instance built from msat_iff_sat.
  3. Which facts are known? sound_forced proves canonical truth from an empty checker ledger. msat_iff_sat preserves and reflects every formula in every Peano model. transport_forced composes them without choice.
  4. Which premise remains empirical? Nature must supply evidence accepted by a separately specified authority for at least one target realization. Lean checks uses of that evidence. Lean does not certify the laboratory or issue the evidence.
  5. What would produce motion without progress? A metadata-only transport square, an equality of a receipt with itself, a family indexed by an empty type, or a target satisfaction relation supplied by the theorem author.
  6. What must remain true after implementation? The empty ledger stays empty, singleton collapse stays excluded, every target quantifier covers its whole carrier, and source truth can be recovered from target validity when every source model has a target expansion.
  7. Does bare carrier equivalence preserve generation history? No. The existing tower module proves that the even naturals receive a TowerGen witness by equivalence. Construction provenance therefore lives in the explicit stage, bundle, and CoreTower fields, not in the bare proposition TowerGen X.
  8. Can a tower receipt itself force a Peano target? No. Tower receipts also cover the signed and fraction stages. Target Peano laws remain a separate proof obligation. This rejects the adverse panel’s unstable proposal to derive all target operations from an arbitrary tower receipt.
  9. Can Lean distinguish honest external evidence from a dishonest True-valued authority? No. The kernel can prove that a false attestation relation has no receipts and that an internal self-issued token does not work for every authority. Authenticity of the authority is an empirical and operational premise.

Frozen Lean contract

The generic layer separates syntax, semantics, and model reduction. The concrete δ layer fixes the sentence map to identity. 

structure Institution where
  Sentence : Type
  Model : Type
  sat : Model → Sentence → Prop

structure Translation (Src Tgt : Institution) where
  sentence : Src.Sentence → Tgt.Sentence
  reduct : Tgt.Model → Src.Model
  satisfaction : ∀ M φ,
    Tgt.sat M (sentence φ) ↔ Src.sat (reduct M) φ

structure ModelExpansion (T : Translation Src Tgt) where
  expand : Src.Model → Tgt.Model
  reduct_expand : ∀ M, T.reduct (expand M) = M

A proof receipt is the checker equation itself. It is not reconstructed after transport. A generated theorem carries the explicit construction object and the empty-ledger certificate. 

structure GeneratedObject where
  stage : TowerStage
  bundle : StageBundle stage
  construction : CoreTower bundle

structure ForcedTheorem where
  derivation : Deriv
  sentence : DFormula
  checked : check [] derivation = some (sentence, Ledger.empty)

structure GeneratedTheorem where
  generated : GeneratedObject
  certificate : ForcedTheorem

Empirical authority is parametric. A family must contain an index as data, so an empty family cannot be constructed. Each member separately supplies its external receipt. 

structure EmpiricalAuthority (I : Institution) where
  Evidence : I.Model → Type
  accepts : ∀ M, Evidence M → Prop

structure EmpiricalReceipt (A : EmpiricalAuthority I) (M : I.Model) where
  evidence : A.Evidence M
  accepted : A.accepts M evidence

structure EmpiricalFamily (A : EmpiricalAuthority I) where
  Index : Type
  witness : Index
  realization : Index → I.Model
  receipt : ∀ i, EmpiricalReceipt A (realization i)

Scoped verdict

THEOREM

Every empty-ledger δ checker theorem is true under msat in every Peano target, and the exact source checker equation remains in the transported result.

THEOREM

Satisfaction equivalence blocks the translator that maps every source sentence to truth. A target-model expansion for each source model implies semantic conservativity.

CONDITIONAL

The phrase “physical truth” is licensed only for a target paired with an accepted empirical receipt from the authority supplied to the theorem.

OPEN

Nature supplies such a receipt for a δ realization. No inhabitant of that claim is added here.

Largest licensed claim. Under an explicitly supplied empirical authority and a nonempty family of receipted Peano realizations, every theorem certified by the δ checker with the empty ledger is true in every family member, with satisfaction reflected back to the canonical source semantics and the original checker receipt unchanged.
Does not license. It does not prove that every Lean type is actual, that every semantic truth is provable, that every mathematical signature has a physical translation, that a bare equivalence preserves native construction history, that a weak constant-trace embedding is faithful, or that a physical realization exists.
